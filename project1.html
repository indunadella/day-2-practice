Calculator Web Application

   

Tech Stack: HTML, CSS, JavaScript

This is a responsive and interactive calculator built using core web technologies. The application allows users to
perform basic arithmetic operations like addition, subtraction, multiplication and division. It is designed with a
clean, user-friendly interface and provides real-time calculation functionality.

Key Features:

1. User-friendly layout with clickable buttons for digits and operations.
2. Real-time input display and calculation results.
3. Clear and reset functionalities for seamless user experience.
4. Responsive design compatible with both desktop and mobile devices.
5. Clean and organized codebase for easy readability and maintenance.

This project helped me strengthen my foundational skills in web development and understand DOM manipulation,
event handling and dynamic styling using Javascript.
